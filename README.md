# Менеджер задач с уведомлениями

Клиентская часть веб-приложения для управления задачами с интегрированной системой уведомлений и напоминаний. Разработано в рамках курсовой работы по дисциплине «Фронтенд-разработка».

## Технологии

- HTML5, CSS3 (БЭМ)
- JavaScript (ES6+), модули, классы
- Vite (сборка)
- localStorage (хранение задач)
- Notification API (уведомления)
- Jest (юнит-тестирование)
- ESLint / Prettier

## Функциональность

- Добавление, редактирование, удаление задач
- Отметка о выполнении
- Фильтрация (все / активные / выполненные)
- Установка срока выполнения и напоминания
- Браузерные уведомления о наступлении времени напоминания
- Отображение задач на сегодня в блоке «Напоминания»
